=====================================
绿黄红
=====================================

MAME 用颜色来稍微标记了一下游戏的模拟状态。
	
	| 绿色，模拟状态 比较好。
	| 黄色，模拟状态 稍差一些。
	| 红色，模拟状态 最差。
	
	也可能不是很准确，有些红色的游戏，运行的效果也可能还不错。

如果用 UI 配合 MAME ，或者，第三方的 MAME ，也可能会用这样的颜色来大致表示一下模拟状态。
