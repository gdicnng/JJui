=====================================
命令行 暂停/停止
=====================================

鼠标点击
	
	好像有些版本的命令行，鼠标 在命令行 点一下，就可能暂停了
	
	容易误操作
	
	我喜欢 关掉 这种功能
	
	对于 MAME 这种窗口程序，似乎没有什么影响
	
	如果用 python.exe 打开 JJui ，命令行中会显示程序运行中输出的文字信息。如果打开它的命令行暂停了，JJui 就卡死了。
	
	如果用 pythonw.exe 打开 JJui ，这种屏蔽了命令行显示，似乎没有什么影响。

Pause 键
	
	暂停
	
	对于 窗口程序 似乎没有什么影响

Ctrl + C
	
	停止


Ctrl + Pause
	
	停止
	
	这个好像更强力一点
