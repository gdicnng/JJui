=====================================
命令行
=====================================

如果不使用命令行的方式运行 MAME ，可以忽略此处。

官网发布的 MAME ，是 windows 操作系统上的版本。
目前，都是免安装的版本，是自解压文件，不用安装，解压缩到某个文件夹里就能使用了。

windows 10 操作系统上自带的命令行有 CMD 和 PowerShell 。

MAME 官网上的说明，很多都是以命令行方式介绍的。
如果查看 官网说明的话，看到命令行指令不用奇怪。
