=====================================
英文名缩写
=====================================

游戏的英文名的缩写很重要，当作 id 来用的。

比如 游戏文件(roms) 的命名方式，就是用的 英文名缩写 。

比如 周边文件的命名方式，也是用的 英文名缩写。
