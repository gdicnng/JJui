==========================================
IPS 补丁
==========================================

第三方的 MAME 比如 MamePlus 模拟器 等，有 IPS 补丁功能。

FBNeo （以前叫 FBAlpha）街机模拟器的话，也有 IPS 补丁功能。

IPS 资源的话，自己可以搜索一下。

官方原版 MAME ，目前，没有 IPS 补丁功能。
	
	可以自己试试 把 IPS 补丁还原为 roms ，
	然后尝试用 MAME 运行。
	这样就算最后能用，也很麻烦。
	
	MxUI 提供了 IPS 补丁功能。
		
		https://www.ppxclub.com/671046-1-1
		
		用 MxUI 配合 MAME 使用。
			
			MxUI 的 IPS 功能，把 IPS 还原为 roms ，然后用 MAME 运行。
			
			这样的话，方便很多。
