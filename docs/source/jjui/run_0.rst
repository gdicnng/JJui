==========================================
运行环境
==========================================
一：

如果使用的已打包好的 exe 程序。
	
	解压程序到一个文件夹，找到 JJui.exe ，运行 JJui.exe 。
		
		注意不要选择 权限比较特殊的 文件夹。

二：

使用源代码。

因为是 python 脚本，所以，可以用直接使用 python 打开 源代码 文件。

	| 如果了解 python ：

	| 	UI 界面用的 python 标准库 tkinter：

	| 		windows 操作系统上，python 官网还有免安装的版本（文件名中有 embed ），这个没有包含 tkinter 库，不用这个。
